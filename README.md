# SignalFlow AI - Project Folder Structure

This folder contains the core components for the SignalFlow AI Micro-SaaS.

## 📂 Structure
- `/pinescript`: TradingView indicators and strategies (v6).
- `/src`: Application source code (Next.js / Supabase logic).
- `/public`: Static assets for the landing page and dashboard.
- `/docs`: Business plan, roadmap, and technical specifications.
