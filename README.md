# SignalFlow AI - Intelligent Investment Copilot 🚀

SignalFlow AI is a premium, global Micro-SaaS that functions as an intelligent investment copilot for retail traders. It translates complex technical data (Smart Money Concepts) and macro-economic factors (DXY/VIX) into actionable, human-readable advice.

## 🌟 Key Features
- **AI Intelligent Narrator**: GPT-4o mini powered engine that explains the "WHY" behind every trade.
- **SMC Detection**: Built-in institutional logic for Order Blocks and Fair Value Gaps.
- **High-Value Alerts**: Humanized notifications delivered via Telegram Premium.
- **Lead Magnet**: Built-in Global Liquidity Monitor for high-conversion user acquisition.

## 🛠️ Tech Stack
- **Frontend**: Next.js 14, Tailwind CSS, Lucide Icons.
- **Backend**: Supabase Edge Functions (Deno/TypeScript), OpenAI API.
- **Signals**: Pine Script (TradingView Webhooks).

## 🚀 Getting Started
1. **Clone the repo**
2. **Install dependencies**: `npm install`
3. **Run local dev**: `npm run dev`
4. **Deploy Edge Functions**: `supabase functions deploy signal-processor`
